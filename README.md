## 开发背景
一直很喜欢用一款叫做URLserch的burpsuite插件，但是感觉bug很多，扫描也不够灵活，所以用新版的motonya-api重构了一下，减少了它的bug，增加了它的威力，同时也随机加了些平时用到的功能
## 开发环境
**jdk-21**

**maven-3.9.x**

**burpsuite-2024.11.2**
